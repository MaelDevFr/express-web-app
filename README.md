```This is a basic web app create with Express JS```

To use this code, you need :

> NodeJs 
> NpmJS
> Visual Studio code ( or an another editor of code )
> A Web Browser

`Step 1 :`

Download the source code and open the folder with your code's editor

`Step 2 :`

Install the packages : npm install

`Step 3 :`

Launch the web app : node .

`Step 4 :`

Go to your web browser and go on : localhost:3000
